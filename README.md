# Project Description
This is a test project to learn about test automation. 
The used technologies are Python and pytest 

# Functions (main.py)
- average: calculates the average of a list of values
- calculate_max: calculates the maximum of a list of values
- calculate_min: calculates the minimum of a list of values
- outliers: calculates value outliers of a list of values based on a given threshold
- is_valid_value: returns feedback if a value is valid or not

# Tests (test_main.py)
Tests of the functions from main.py including edge cases

# How to execute the tests
pip install pytest
pytest test_main.py
